# 🍫 巧克力对半分（等份切分）

一个面向小朋友的网页益智游戏：把一整块 **M × N 格** 的巧克力，沿着格子之间的**缝隙**画线，
切成**大小、形状一模一样**、而且**各自连在一起**的两半。

同一种巧克力会同时摆出**好几块**，方便孩子在一个屏幕上**同时尝试多种不同的分法**，比比看谁分得更巧。

🎮 在线试玩：`https://xukaigao.github.io/splitChocolate/`

## 怎么玩

1. 在巧克力的**缝隙**上点一下，就把那条缝「切开」；再点一下可以取消。
2. 也可以**按住拖动**，沿着缝隙一次画一长条。
3. 目标：把整块巧克力切成**恰好两块**，两块**格子数一样多**、**形状一模一样**（可以旋转或翻面对上），且每块都连在一起。
4. 成功后这一块会高亮、撒小星星 ✨。

> 顶部可以选 **行数 / 列数 / 同时摆几块**。格子总数必须是偶数才可能平分（奇数会有提示）。

## 计分与目标

- **已分成功**：当前有几块巧克力被成功分成了合格的两半。
- **不同的分法 🏆**：在所有成功的块里，统计出现了多少种**不同形状**的分法。**按形状去重**：把切下来的半块翻转、旋转后能重合的，算同一种。
- **目标提示**：顶部会显示「🎯 这个尺寸一共能切出 N 种不同形状，已找出 Y 种」，孩子可以照着这个目标把所有形状都找出来；集齐时会有 🎉 提示。
  - 块数仍由你手动选，与形状总数无关，多摆几块只是方便同时尝试。
  - 总数是程序实时枚举算出来的；尺寸较大时形状极多，会改为提示「能切出非常多种形状，尽情探索吧」而不显示具体数字。
- **鼓励弹窗**：当找出的形状达到**一半**、以及**全部找齐**时，会弹出祝贺，并随机送上一句表扬的话（共 20 句，肯定孩子的努力与思考）。尺寸过大、没有精确总数时，则每找到 5 种鼓励一次。

## 规则判定（实现说明）

把每个格子当作节点，相邻两格之间若缝隙**没被切开**就连通，用洪水填充求连通块：

- 必须**恰好 2 块**（多于 2 块说明切碎了，少于 2 块说明还没切通）。
- 两块**格子数相等**（各占总数的一半）。
- 两块**全等**：把一块在 8 种二面体变换（旋转 0/90/180/270 × 是否镜像）下归一化，能和另一块重合即算「一模一样」。
- 因为用 4 连通求连通块，所以「每半块都连在一起」自动满足。

## 技术栈

- 纯 **HTML + CSS + 原生 JavaScript**，无框架、无构建步骤。
- **SVG** 绘制巧克力格子与切线，缝隙用透明热区接收点按 / 拖动（Pointer Events，支持触屏）。

## 文件结构

```
splitChocolate/
├── index.html      # 页面结构 + 控制面板
├── css/style.css   # 巧克力主题样式、响应式
└── js/game.js      # 渲染、切割交互、连通块与全等判定
```

## 本地运行

直接用浏览器打开 `index.html` 即可（无需构建、无依赖），或起一个本地静态服务器：

```bash
python -m http.server 8000   # 然后访问 http://localhost:8000
```

## 部署为 GitHub Pages

推到 `xukaigao/splitChocolate` 仓库后，在 **Settings → Pages** 里选 `Deploy from a branch`，
分支 `main`、目录 `/ (root)`，保存。约 1~2 分钟后访问 `https://xukaigao.github.io/splitChocolate/`。
（免费版需要仓库为 public 才能发布 Pages。）

随后在游戏厅首页 `xukaigao.github.io` 的 `index.html` 里把这个游戏加进 `GAMES` 数组即可。
